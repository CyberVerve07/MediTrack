# **App Name**: MediTrack Pro

## Core Features:

- Patient Profile Management: Create, update, and manage patient profiles, including personal details, medical history, and contact information.
- Health Status Tracking: Record and track patient health status, vital signs (BP, sugar, blood levels), and ongoing treatments. Allow nurses to log all essential patient data.
- Medication Management: Log administered medications, dosage, and time of administration.
- Test Report Management: Upload, store, and view patient test reports. Allow categorization of reports based on test type.
- Admission/Discharge Tracking: Track patient admission and discharge dates.
- Automated Billing: Automatically generate patient bills based on treatments, medications, and stay duration.
- AI-Powered Insight Generation: Use generative AI to generate improvement notes based on entered patient data. The AI feature will function as a tool that can intelligently incorporate relevant information from historical records and real-time health data to create targeted notes that give precise, context-aware guidance.

## Style Guidelines:

- Primary color: Deep calming blue (#3F51B5) to convey trust and stability, drawing inspiration from classic healthcare aesthetics. This hue aligns with conveying trust, security, and cleanliness.
- Background color: Light pale blue-gray (#E8EAF6), creating a clean and professional canvas that enhances readability and user focus.
- Accent color: Soft teal (#4DB6AC) for interactive elements and highlights, providing a modern touch and easy visual navigation.
- Body and headline font: 'PT Sans', a versatile sans-serif font to ensure clarity and readability across various screen sizes.
- Use universally recognized, clean, and simple icons for all key functions and data points to facilitate intuitive navigation.
- Design a clear, tab-based layout to ensure nurses can quickly switch between patient profiles, health data, and billing information.
- Use subtle transitions and feedback animations for data updates and interactions to provide a smooth and responsive user experience.
